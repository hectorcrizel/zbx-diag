# Zabbix Proxy Diagnostic Tool

![Bash](https://img.shields.io/badge/Bash-Script-4EAA25.svg)
![Zabbix](https://img.shields.io/badge/Zabbix-Troubleshooting-red.svg)

A lightweight Bash utility designed to diagnose network and communication issues between Zabbix Proxies and Servers.

## 🚀 Key Features

*   **Connectivity Tests:** Rapid verification of proxy-to-server communication.
*   **VPN Optimized:** Ideal for troubleshooting complex multi-site deployments over VPN tunnels.
*   **Lightweight:** A single shell script (`zbx_diag.sh`) with no heavy dependencies.

## 🛠️ Tech Stack

*   Bash / Shell Script
*   Zabbix API (optional connectivity check)
